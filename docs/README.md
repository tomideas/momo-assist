# Momo 說明文件（HTML）

本目錄是 **Hii~ Momo: AI Assist** 的使用者說明書，與擴充功能原始碼同屬 [momo-assist](https://github.com/tomideas/momo-assist) repo。

## 閱讀方式

推送 `main` 後，GitHub Actions 自動部署至 GitHub Pages：

👉 **https://tomideas.github.io/momo-assist/**

首次啟用：GitHub repo → **Settings** → **Pages** → Source 選 **GitHub Actions**。

## 本機預覽

```bash
python3 -m http.server 8765 --directory docs
# 瀏覽器開 http://127.0.0.1:8765/
```
