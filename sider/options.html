聯網搜尋設定

在側邊欄開啟聯網搜尋後，僅在訊息看起來需要即時網路資訊時才會搜尋（例如新聞、價格、天氣、明確「上網查／搜尋」、訊息內網址等），並將結果作為上下文提供給 AI；未開啟時，只有明確「上網查／搜尋」等訊息會自動觸發搜尋。

開啟時只使用搜尋摘要，關閉時會額外訪問搜尋結果的頁面取得完整內容（較慢但更精確）。

開啟時，若訊息中包含網址，會自動抓取該頁面的內容並提供給 AI。

開啟時，每次對話預設啟用聯網搜尋。

到 brave.com/search/api 免費註冊即可取得（每月 2000 次免費搜尋，無需信用卡）。

到 tavily.com 免費註冊即可取得 API Key（每月 1000 次免費搜尋）。

引用頁面內容設定

選擇想擷取的區塊。「Markdown 智能提取」使用 Mozilla Readability 智能識別主要內容，自動過濾導航、廣告、側邊欄等，並轉換為 Markdown 格式，讓 AI 更容易理解，節省 token。

設定每次引用頁面時發送給 AI 的最大字符數（建議：20000-100000），包含頁面標題、URL、正文等全部內容。數值越大，可捕獲的內容越多，但也會消耗更多 token。Amazon 等複雜頁面建議設置 100000 以上。

✨ 智能捕獲：插件會自動檢測虛擬滾動網站（如 Amazon、Twitter、Reddit 等），並使用智能滾動模式自動捕獲完整內容。點擊「引用頁面」按鈕後，頁面會自動滾動並累積所有內容（需時 10-30 秒），捕獲完成後內容會自動添加到對話中。可多次點擊重新捕獲最新內容。

Server setup (required for remote gateway)

Momo connects with Token Only (no device pairing). For remote gateways, configure ~/.openclaw/openclaw.json like below — then use the same token in Momo.

  1. Edit ~/.openclaw/openclaw.json (example — replace placeholders): 
    {
  "gateway": {
    "port": 18789,
    "mode": "local",
    "bind": "lan",
    "controlUi": {
      "allowedOrigins": [
        "http://YOUR_HOST:PORT",
        "null"
      ],
      "allowInsecureAuth": true,
      "dangerouslyDisableDeviceAuth": true
    }
  },
  "auth": {
    "mode": "token",
    "token": "<your-gateway-token>"
  }
}

    YOUR_HOST:PORT — use the same host/port as Momo’s Gateway URL (ws://YOUR_HOST:PORT; Momo sends http://YOUR_HOST:PORT as Origin). Include "null" for Chrome extension fallback. gateway.port must match your listen port. Set auth.token to your secret; paste the same value into Momo Gateway Token.

  2. Restart or reload the gateway for changes to take effect.
  3. In Momo: Gateway URL ws://YOUR_HOST:PORT (or http://, auto-converted), same Gateway Token, then Connect → pick a Session.
▲

Anyone with the token gets full access. Only use on local/trusted networks. Gateway must support protocol v4.

Hermes API Server setup

和 OpenClaw 一樣，Hermes 需要先在 Hermes 的 .env 開啟 API Server，然後重啟 Hermes gateway。

  1. 在 Hermes .env 修改： 
    API_SERVER_ENABLED=true
API_SERVER_HOST=0.0.0.0
API_SERVER_PORT=8642
API_SERVER_KEY=<your-secret-key>
API_SERVER_CORS_ORIGINS=*

    請把 <your-secret-key> 換成你自己的密碼，並在 Momo 的 API Server Key 欄填入同一個值。

  2. 重啟 Hermes gateway，讓 .env 變更生效。
  3. 在 Momo 填入 Base URL，例如 http://127.0.0.1:8642/v1 或你的遠端 URL，然後按 Connect。
▲

API_SERVER_CORS_ORIGINS=* 是 Chrome extension / 遠端連線最簡單可用設定；公開服務請改用更嚴格的可信任 origin。
